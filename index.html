HOW THE ENEMY THINKS

BFS PATHFINDING ALGORITHM

The enemy uses Breadth-First Search (BFS) to compute the absolute shortest path to you on every move tick.

It expands outward through all valid walkable neighbors until it finds you. Result: it never gets lost, never wastes moves, never gives up.

# Enemy AI — BFS next step (Python source) def bfs_next(start, target): if start == target: return start q = deque([[start]]) vis = {start} while q: path = q.popleft() cur = path[-1] if cur == target: return path[1] if len(path) > 1 else start for nb in get_neighbors(cur): if nb not in vis: vis.add(nb) q.append(list(path) + [nb]) return start
